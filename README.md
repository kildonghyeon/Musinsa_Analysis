# 무신사 랭킹 데이터 분석
<img src="/images/musinsaLogo.jpeg" width="80%"/>

## 개요
평소 관심이 많던 대한민국 최대 규모의 온라인 편집숍인 무신사 사이트의 상품 랭킹페이지를  
크롤링, 데이터 분석을 함으로써 구매 시 어떤 브랜드가 가장 합리적인지를 판단하는 데 도움이  
될 수 있을 지 그래프 및 표로 볼 수 있다면 좋겠다 생각하고 시작하게 된 데이터 분석 프로젝트입니다.

# 시각화

ggplot을 사용한 막대그래프  
<img src="/output/brandTop20.png" width="68%"/>  
matplotlib을 사용한 파이차트  
<img src="/output/brandTop20(pie).png" width="68%"/>  
두 그래프, 파이차트를 보면 무신사 스탠다드 상품수가 전체 비중에 굉장히 많은걸 확인 할 수 있습니다.  
또, 상위권 브랜드도 전체에서 적지않는 비중을 가지고 있다는 것도 알 수 있습니다.  
이렇게 많은 비중을 차지하고 있는 상위권 브랜드들의 가격대는 어떻게 형성 되어있을까 하여  
상위권 브랜드의 최소, 평균, 최대 금액을 비교해보았습니다.

## 랭킹페이지 내 상품수 상위권 브랜드 최소, 평균, 최고가격 비교
<img src="/output/brandTop20(price)_1.png" width="100%"/>  
최소금액과 최대금액의 가격차가 너무 큰 브랜드는 수치가 제대로 확인이 안되는 문제가 있었습니다.  
그래서 전체 가격에 25%, 50%, 75%로 다시 그래프를 만들어 확인해보았습니다.  
<img src="/output/brandTop20(price)_1.png" width="100%"/>  
최소, 평균, 최대금액 비교 그래프보다 확인이 더 잘 되는 것을 볼 수 있습니다.  
상위권브랜드 중에서는 무신사 스탠다드, 어반드레스는 가격대가 상위권에서 비교적 저렴한 편이고  
유명 아웃도어 브랜드의 경우 전체적으로 가격대가 좀 있는 편이라는 것을 보여주는 그래프였습니다.  
최소, 평균, 최대 그래프를 보면 가격차가 굉장히 컸던 것을 확인할 수 있는데 
상위권 브랜드 가격별 좋아요 수 평균을 보면 브랜드마다 어떤 가격대의 상품이 반응이 좋은지 볼 수 있을 것 같아  
그래프로 만들어 비교해보았습니다.
<img src="/output/brandTop20(price_like).png" width="100%"/> 

## 랭킹페이지 내 평균 좋아요 수 상위권 브랜드 최소, 평균, 최대 가격비교
상품수 기준 상위권 브랜드말고도 랭킹페이지 내 등록된 상품들 중 평균 좋아요 수가 높은 브랜드는  
어떤 브랜드가 있을 지, 상품수 상위권 브랜드와 동일한 브랜드가 대부분일 지 궁금증이 생겨 후처리  작업 후 최소, 평균, 최고 금액을 비교해보았습니다.  
<img src="/output/likeTop20(price).png" width="100%"/>  
제일 먼저 상품 수 상위권 브랜드들과는 전혀 다른 브랜드들이 순위를 차지하였습니다.
가격과 최소, 최고 금액의 차이는 하 아카이브 제외 상품수 상위권 브랜드의 비해
저렴한 가격대 그리고 높지 않는 편차를 보이는 것을 확인 할 수 있었습니다.

## 랭킹페이지 내 평균 좋아요 수 상위권 브랜드 가격대 별 평균 좋아요 수 비교
그렇다면 평균 좋아요 수 상위권 브랜드는 어느 가격대가 소비자에게  
관심이 많을 지 확인하기 위해 가격대 별 평균 좋아요 수를 비교해보았습니다.  
<img src="/output/likeTop20(price_like).png" width="100%"/>  
5만원 이하인 상품들 위주로 좋아요 수가 평균적으로 높다는 것으로 보아,  
대체적으로 소비자들은 저렴하고 가성비가 좋은 상품쪽에 관심을 많이 갖는다는 걸  
확인할 수 있었습니다.
