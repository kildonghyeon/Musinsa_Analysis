## 개요
평소 관심이 많던 대한민국 최대 규모의 온라인 편집숍인 무신사 사이트의 상품 랭킹페이지를<br/>
크롤링, 데이터 분석을 함으로써 구매 시 어떤 브랜드가 가장 합리적인지를 판단하는 데 도움이<br/>
될 수 있을 지 그래프 및 표로 볼 수 있다면 좋겠다 생각하고 시작하게 된 데이터 분석 프로젝트입니다.<br/>
