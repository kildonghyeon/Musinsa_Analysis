# 무신사 랭킹 데이터 분석
<img src="/images/musinsaLogo.jpeg" width="80%"/>

## 개요
평소 관심이 많던 대한민국 최대 규모의 온라인 편집숍인 무신사 사이트의 상품 랭킹페이지를  
크롤링, 데이터 분석을 함으로써 구매 시 어떤 브랜드가 가장 합리적인지를 판단하는 데 도움이  
될 수 있을 지 그래프 및 표로 볼 수 있다면 좋겠다 생각하고 시작하게 된 데이터 분석 프로젝트입니다.

두 그래프, 파이차트를 보면 무신사 스탠다드 상품수가 전체 비중에 굉장히 많은걸 확인 할 수 있습니다.  
또, 상위권 브랜드도 전체에서 적지않는 비중을 가지고 있다는 것도 알 수 있습니다.  
이렇게 많은 비중을 차지하고 있는 상위권 브랜드들의 가격대는 어떻게 형성 되어있을까 하여  
상위권 브랜드의 최소, 평균, 최대 금액을 비교해보았습니다.

## 상위권 브랜드의 최소, 평균, 최대 금액 비교

최소금액과 최대금액의 가격차가 너무 큰 브랜드는 수치가 제대로 확인이 안되는 문제가 있었습니다.  
그래서 전체 가격에 25%, 50%, 75%로 다시 그래프를 만들어 확인해보았습니다.

최소, 평균, 최대금액 비교 그래프보다 확인이 더 잘 되는 것을 볼 수 있습니다.  
상위권브랜드 중에서는 무신사 스탠다드, 어반드레스는 가격대가 상위권에서 비교적 저렴한 편이고  
유명 아웃도어 브랜드의 경우 전체적으로 가격대가 좀 있는 편이라는 것을 보여주는 그래프였습니다.
